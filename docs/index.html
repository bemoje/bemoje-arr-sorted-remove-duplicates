arrSortedRemoveDuplicates

Remove all duplicate elements in a sorted array, leaving only uniques.

arrSortedRemoveDuplicates(arr: Array, compare: (comparator | object)?): arr
Parameters
arr (Array) The sorted array
compare ((comparator | object)?)
Name Description
compare.numeric boolean (default false) Sort numerically. Defaults to lexicographic/alphabetic sort.
compare.descending boolean (default false) Sort in descending order. Defaults to ascending order.
compare.array boolean (default false) Sort arrays. Nested arrays are also compared recursively.
compare.by (number | string | getter) (default undefined) Sort by either array index, a callback(element): any - or by object keys with dot-notation support.
Returns
arr:

comparator

Comparator function callback definition.

comparator(a: any, b: any): number

Type: Function

Parameters
a (any) The first value to compare
b (any) The second value to compare
Returns
number: A negative number if a > b, a positive number if a < b, 0 otherwise.

Value-getter function callback definition.

getter(a: any): any

Parameters
